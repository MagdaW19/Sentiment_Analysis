# Sentiment_Analysis
Sentiment Analysis using Amazon Reviews for products from category Electronics.

# Original Data 
## Amazon Review Data (2018)
Source: https://nijianmo.github.io/amazon/index.html <br>
Description: This Dataset is an updated version of the Amazon review dataset released in 2014.<br>
As in the previous version, this dataset includes reviews (ratings, text, helpfulness votes), <br>
product metadata (descriptions, category information, price, brand, and image features), and <br>
links (also viewed/also bought graphs).<br>
## Original Paper
Justifying recommendations using distantly-labeled reviews and fined-grained aspects <br>
Jianmo Ni, Jiacheng Li, Julian McAuley <br>
Empirical Methods in Natural Language Processing (EMNLP), 2019 <br>
